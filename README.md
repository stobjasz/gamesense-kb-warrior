# gamesense-kb-warrior

**gamesense-kb-warrior** is a compact idle fantasy game for SteelSeries GameSense-compatible devices. It transforms your keyboard display into a miniature battle scene where a warrior clashes with monsters using animated pixel graphics tailored for a **128x40 px** screen.

The application currently supports **Windows** and runs quietly in the background with a **system tray icon**, making it easy to keep active without interrupting your workflow. You can close it from the tray menu at any time or instantly exit with **Ctrl+Alt+Backspace**.

## Features

- Idle game experience built for SteelSeries devices
- Animated battle visuals optimized for **128x40 px** displays
- Lightweight background app with tray integration
- Quick keyboard shortcut to quit: **Ctrl+Alt+Backspace**
- Windows-only support for now
