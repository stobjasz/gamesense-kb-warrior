# gamesense-kb-warrior
Simple idle game for SteelSeries devices.  

Will work on Windows only for now and generates graphics for 128x40 px.  

Creates a system tray icon with an option to quit the application.  
You can also quit it by pressing Ctrl+Alt+Backspace

Monsters: https://butterhands.itch.io/doomsphere-charset  
Warior: https://daliszito.itch.io/1-bit-medieval-character-pack
